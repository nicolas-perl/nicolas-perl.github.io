Bauernschnapsen

Based on the classic Austrian card game Bauernschnapsen (also known as 4er-Schnapsen), this game brings strategic team-based trick-taking fun to your device. You can play with smart AI opponents and experience authentic Austrian gameplay.

  • Research & Prototyping
    The first step was to get familiar with the gameplay and its many different variations. The game is played with a 20-card deck, where each player has 5 cards. Playing in teams of two, the goal is to win 24 points by playing different strategies and winning multiple individual rounds.

    Comparing different implementations and apps focusing on the two-player version of this game, I started to highlight common features and layouts and created first hand-drawn wireframes.
  • Development
    This being my first Flutter project, I started with the implementation of the game logic and by getting familiar with the framework and language itself. Using version control software from the get-go, I started with a terminal-based version to get familiar with the game logic and its rules.

    Focusing solely on the "normal mode" game logic first, I created a fist UI prototype that incorporated the proposed wireframes as a guide. The major challenges were the implementation of the 20/40 pair calls and the game state management.
  • After the logic was implemented and bug-free, I started to implement the backend. The approach was to use a local drift database to store the game state and player data when offline. This way, the game can be played offline and the data is stored locally. If the user has internet connection, the data is synced with a remote Supabase database.
  • Adding Features & Final Prototype
    Over several iterations a more refined version was created including a login, music and sound effects. Adding music, using a fitting font an visuals truly completed the app.

    The next step is to do some user testing, get feedback and make the app available on the Play Store / App Store. A German language setting as well as multiplayer mode would also be a great addition.
